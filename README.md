##  CI/CD Pipeline
This project uses a CI/CD pipeline implemented using GitHub Actions.

The pipeline automatically:
- Builds the project
- Runs tests
- Deploys the application (if configured)

Whenever code is pushed to the repository, the pipeline is triggered automatically to ensure code quality and smooth deployment.
